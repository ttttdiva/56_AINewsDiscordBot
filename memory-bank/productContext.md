# Product Context

更新日: 2026-03-11

## 対象ユーザー

- 毎日 AI ニュースを追いたい個人ユーザー
- Discord を普段の情報ハブとして使っているユーザー

## 解決したい痛み

- X での情報収集が散らかりやすい
- 毎日似た検索を繰り返している
- 重要投稿だけ見たいが、手動での選別が面倒
- 見つけたニュースを別の場所に転記するのが手間

## 期待する UX

- 毎日決まった時刻に 1 通届く
- 長すぎず、ニュースの核心だけ読める
- 参照元投稿にすぐ飛べる
- 誤爆や重複投稿が少ない

## MVP の出力イメージ

- 日付見出し
- 全体サマリ 2-4 行
- 主要トピック 3-7 件
- 各トピックに短い要約と投稿 URL
- 必要なら補足欄
