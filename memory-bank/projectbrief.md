# Project Brief

更新日: 2026-03-11

## Why

毎日 AI 関連ニュースを追う作業を手動でやると、検索、取捨選択、要約、Discord 投稿が分断されていて面倒です。これを 1 本の Bot にまとめ、毎日決まった時刻に Discord に届く状態を作るのが目的です。

## What

- Grok API の `x_search` を使って X 上の AI 関連話題を収集する
- 重要度の高い投稿だけを選び、短い日本語要約にまとめる
- Discord の指定チャンネルへ毎日 1 回投稿する
- 過去に送信した投稿の重複再送を防ぐ

## Definition Of Done

- Bot が毎日 1 回自動実行される
- 収集から Discord 投稿まで無人で完了する
- 失敗時にログを見れば原因が分かる
- 同じ投稿 URL または同じ投稿 ID を二重送信しない
- セットアップ手順がドキュメント化されている

## Non Goals

- 会話型 Discord Bot
- 音声チャット
- Web 管理画面
- 手動承認ワークフロー
- 複数チャンネルへの配信ルーティング
